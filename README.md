# BySykkel
Retrieve city bicycle stands information for public API


#How to use

## Windows 

### Requirements 
#### Python 3.6 and up

### Usage

python ride/main.py 


## Ubuntu on Windows 10
### Requirements

apt-get install python3
apt-get install python3-tk

install XMing on windows

echo "export DISPLAY:=0" >> ~/.bashrc

and only then

python3 ride/main.py
